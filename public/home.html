Coding Project: My First Foray Into NodeJS

Thank you for taking the time to look at my first nodejs, express experiment. After getting past the initial learning curve, I completely understand why nodejs is so popular!

Some things I would like to point out about this project:

  1. "Trading Action" is the central webpage. Here, I used ejs to create a table that refreshes dynamically with the latest Bitcoin transaction data (all data is provided by Bitfinex). The refresh script can be paused in order to look more closely at the data, then resumed.
  2. The user can choose the maximum number of rows of data to display at a time. I have provided the options of 10, 50, 100, or 200, but theoretically any number could be chosen and the page would adjust.
  3. I added an animation to notify the user every time the flow of data is resumed.
  4. The sidebar allows the user to request different pages from the express server.
